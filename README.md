# *Objetivo*

Este relatório tem o objetivo de realizar o acompanhamento diário das aprovações do cenário 3, e com isso ajudar na recomposição da base do sistema Benner e garantir o correto preenchimento das informações por nosso parceiro SOS Telecom.

As atualizações são realizadas 5 vezes ao dia, sendo duas de manha e tres na parte da tarde.
